# 若依框架知识整合

## 一、自定义注解

### 1 @Log

![image-20241130172507178](assets/image-20241130172507178.png)

加在方法和属性上，如果用户对其操作时，会在执行前后对``sys_oper_log``表进行插入操作记录和操作情况

### 2 @RateLimiter

![image-20241202160158903](assets/image-20241202160158903.png)

加在方法上，在指定时间内限制访问次数

### 3 @DataSource

![image-20241202160904229](assets/image-20241202160904229.png)

加在方法上或者类、接口上，指定访问的数据源是主库数据还是从库数据
